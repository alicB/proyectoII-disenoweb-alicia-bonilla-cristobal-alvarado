How to view the created projects

When you are log in, press the project button to go to the project menu.

When you are in the project menu, you can see the created projects.

How to create projects

Press the button new to create a new project and fill in the spaces with the information that you want.

How to edit projects

Press the button edit and fill in the spaces with the information that you want to change. Don't forget to save the changes.

How to delete projects

Press the button delete, it will ask you if you want to delete project.

How to view the project board

In the project card, you can see some buttons with different options. Press the button View. If the project doesn't have a board, you will need to create it.

How to create a board

Press the button new to create a new board and fill in the spaces with the information that you want.

How to edit boards

Press the button edit and fill in the spaces with the information that you want to change. Don't forget to save the changes.

How to delete boards

How to view the task's board

In the board card, you can see some buttons with different options. Press the button View. If the board doesn't have a task, you will need to create it.

How to view the task's board

In the board card, you can see some buttons with different options. Press the button View. If the board doesn't have a task, you will need to create it.

How to create a task

Press the button new to create a new task and fill in the spaces with the information that you want.

How to edit a task

In the task card, you can see some buttons with different options. Press the button Edit.

How to add a user

Press the icon person, it will appears a box, add the user you want to add and that's all. Don't forget to press the button send invitation or cancel if you don't want to send it.

How to delete a task

Press the button delete, it will ask you if you want to delete task.

How to access to the teams
